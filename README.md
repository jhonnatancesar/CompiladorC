# Compiladores GRUPO 1

Utilização do Programa.

```console
$ make clean
$ make 
$ ./interpretador

Bison, Flex etc
$ wget http://bnfc.digitalgrammars.com/download/bnfc-2.9.3-linux-x86_64.bin                                                                                                   
$ chmod +x bnfc-2.9.3-linux-x86_64.binary
$ ./bnfc-2.9.3-linux-x86_64.binary --version
$ ./bnfc-2.9.3-linux-x86_64.binary --c -m --force Jair.cf
$ flex Jair.l
$ bison -dy Jair.y
$ ls
$ make
```
